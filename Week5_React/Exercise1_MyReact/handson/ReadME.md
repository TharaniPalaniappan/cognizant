# Exercise 1 — myfirstreact

## Setup
This folder only contains `src/App.js` (the file you need to edit).
To run the full app, create it fresh and drop this file in:

```bash
npm install -g create-react-app
npx create-react-app myfirstreact
cd myfirstreact
# replace src/App.js with the one in this folder
npm start
```

Open http://localhost:3000 — you should see the heading
"Welcome the first session of React".
